# Rum

A Windows game launcher for Apple Silicon Macs.

Rum lets you run Windows games on your Mac using Wine and Metal-based GPU translation layers (DXMT/DXVK), with a clean native macOS UI.

## Download

[Download Rum 1.0](https://github.com/RumDeveloper/Rum/releases/download/v1.0/Rum.dmg)

## Installation

1. Download `Rum.dmg`
2. Open the DMG and drag Rum to your Applications folder
3. Right-click Rum → Open (first launch only, bypasses Gatekeeper)
4. Follow the setup wizard

## Requirements

- Apple Silicon Mac (M1 or later)
- macOS 14 or later
- Wine (Rum will guide you through installation)
